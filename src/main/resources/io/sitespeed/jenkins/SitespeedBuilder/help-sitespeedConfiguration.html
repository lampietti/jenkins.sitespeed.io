Configure how you want to run sitespeed. Check here which parameters you can use.
Example test the url sitespeed.io, collect Navigation Timing data, use Chrome and test each page five times: -u http://www.sitespeed.io -c true -z "-b chrome -n 5"
By default the resultBaseDir (where the output is placed with the parameter -r) is set to Jenkins workspace dir + /sitespeed-result/ and you can't change them running from Jenkins.
